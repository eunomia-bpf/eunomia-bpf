# eserver

## Start
```console
$ sudo ./eserver
Server start at 127.0.0.1:8527
```

```console
$ ./eserver --help
Usage: eserver [OPTIONS]

Options:
  -c, --config <CONFIG>  
  -s, --secure           
  -p, --port <PORT>      server port [default: 8527]
  -a, --addr <ADDR>      [default: 127.0.0.1]
  -h, --help             Print help
```
