# ecli-server

## Start
```console
$ sudo ./ecli-server
Server start at 127.0.0.1:8527
```

```console
$ ./ecli-server --help
Usage: ecli-server [OPTIONS]

Options:
  -p, --port <PORT>      Port to bind [default: 8527]
  -a, --addr <ADDR>      Address to bind [default: 127.0.0.1]
  -h, --help             Print help
```
